Last modified: 03.04.2012

The implementation of ARC4 steam cipher consists of the following files: 
1) ARC4_PRG.cal: implements the PRG used in the ARC4.
2) Top_ARC4.xdf: implements the ARC4 stream cipher, which can be used to encrypt/decrypt the plaintext/ciphertext.
An example showing the usage of this stream cipher can be found in the "Testbeds".
